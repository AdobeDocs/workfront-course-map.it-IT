# Gestione Coda

Il percorso di apprendimento di Workfront One per la gestione delle code è stato migrato nel corso [Gestione delle code](https://experienceleague.adobe.com/docs/courses/using/workfront-u-1-2022-1-request-queues-1-0.html){target="_blank"} su Experience League.
